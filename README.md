# Master-Fees101

A web application for paying school fees.

## Tech Stack

*   **Frontend:**
    *   React
    *   Vite
    *   Tailwind CSS
    *   Radix UI
    *   Lucide React
    *   Framer Motion
    *   Sonner
    *   jsPDF
*   **Backend & Deployment:**
    *   Firebase
    *   Supabase
    *   Hono

## Dependencies

To install the required dependencies, run the following command:

```bash
npm install
```

This will install all the necessary packages listed in the `package.json` file.
